EXPERIMENT 8: CONDUCTION OF UNIT TESTS USING BASIS PATH TESTING
================================================================================

OBJECTIVE:
To execute unit tests using Basis Path Testing methodology, targeting all
independent paths in critical modules, and provide comprehensive documentation
of test execution, coverage, and cyclomatic complexity analysis.

UNIT TESTING APPROACH: Jest + Basis Path Testing (McCabe's Strategy)

KEY PRINCIPLES:
  • Identify all decision points in each method
  • Calculate cyclomatic complexity (M)
  • Create minimum M test cases per method
  • Achieve 100% decision and path coverage
  • Document control flow for each path

TEST CASE 8.2: INVENTORY SERVICE - STOCK ADJUSTMENT UNIT TEST
Objective: Test inventory stock adjustment logic
Test Type: Unit Test (Isolated)
Scope: InventoryService.adjustStock() method
Dependencies Mocked: InventoryRepository, BookRepository, AuditService
Test Code:
```typescript
describe('InventoryService', () => {
  let inventoryService: InventoryService;
  let mockInventoryRepo: jest.Mocked<InventoryRepository>;
  let mockBookRepo: jest.Mocked<BookRepository>;
  let mockAuditService: jest.Mocked<AuditService>;

  beforeEach(() => {
    mockInventoryRepo = {
      update: jest.fn(),
      findByBookId: jest.fn(),
      recordAdjustment: jest.fn(),
    };
    mockBookRepo = {
      findById: jest.fn(),
      update: jest.fn(),
    };
    mockAuditService = {
      log: jest.fn(),
    };
    inventoryService = new InventoryService(
      mockInventoryRepo,
      mockBookRepo,
      mockAuditService
    );
  });

  describe('adjustStock', () => {
    test('should increment stock on positive adjustment', async () => {
      const bookId = 1;
      const adjustment = { quantity: 50, reason: 'Restock', reference: 'PO-001' };

      mockBookRepo.findById.mockResolvedValue({
        id: bookId,
        currentStock: 100,
      });
      mockInventoryRepo.update.mockResolvedValue({
        id: 1,
        quantity: 150,
      });

      const result = await inventoryService.adjustStock(bookId, adjustment);

      expect(result.quantity).toBe(150);
      expect(mockInventoryRepo.recordAdjustment).toHaveBeenCalledWith(
        expect.objectContaining({ quantity: 50 })
      );
    });

    test('should decrement stock on negative adjustment', async () => {
      const bookId = 1;
      const adjustment = { quantity: -30, reason: 'Damage', reference: 'ADJ-001' };

      mockBookRepo.findById.mockResolvedValue({
        id: bookId,
        currentStock: 100,
      });
      mockInventoryRepo.update.mockResolvedValue({
        id: 1,
        quantity: 70,
      });

      const result = await inventoryService.adjustStock(bookId, adjustment);

      expect(result.quantity).toBe(70);
      expect(mockAuditService.log).toHaveBeenCalled();
    });

    test('should prevent negative stock', async () => {
      const bookId = 1;
      const adjustment = { quantity: -150, reason: 'Sale', reference: 'ORDER-001' };

      mockBookRepo.findById.mockResolvedValue({
        id: bookId,
        currentStock: 100,
      });

      await expect(
        inventoryService.adjustStock(bookId, adjustment)
      ).rejects.toThrow('Insufficient stock');
    });

    test('should require adjustment reason', async () => {
      const bookId = 1;
      const adjustment = { quantity: 50, reason: '', reference: 'PO-001' };

      await expect(
        inventoryService.adjustStock(bookId, adjustment)
      ).rejects.toThrow('Reason required');
    });

    test('should create audit trail for adjustment', async () => {
      const bookId = 1;
      const adjustment = { quantity: 50, reason: 'Restock', reference: 'PO-001' };

      mockBookRepo.findById.mockResolvedValue({
        id: bookId,
        currentStock: 100,
      });
      mockInventoryRepo.update.mockResolvedValue({
        id: 1,
        quantity: 150,
      });

      await inventoryService.adjustStock(bookId, adjustment);

      expect(mockAuditService.log).toHaveBeenCalledWith(
        expect.objectContaining({
          action: 'STOCK_ADJUSTED',
          details: expect.stringContaining('50'),
        })
      );
    });

    test('should validate book exists', async () => {
      const bookId = 999;
      const adjustment = { quantity: 50, reason: 'Restock', reference: 'PO-001' };

      mockBookRepo.findById.mockResolvedValue(null);

      await expect(
        inventoryService.adjustStock(bookId, adjustment)
      ).rejects.toThrow('Book not found');
    });

    test('should handle zero adjustment', async () => {
      const bookId = 1;
      const adjustment = { quantity: 0, reason: 'Recount', reference: 'REC-001' };

      mockBookRepo.findById.mockResolvedValue({
        id: bookId,
        currentStock: 100,
      });

      const result = await inventoryService.adjustStock(bookId, adjustment);

      expect(result.quantity).toBe(100);
      expect(mockInventoryRepo.update).toHaveBeenCalled();
    });

    test('should record reference for traceability', async () => {
      const bookId = 1;
      const reference = 'PO-12345';
      const adjustment = { quantity: 50, reason: 'Restock', reference };

      mockBookRepo.findById.mockResolvedValue({
        id: bookId,
        currentStock: 100,
      });
      mockInventoryRepo.update.mockResolvedValue({
        id: 1,
        quantity: 150,
      });

      await inventoryService.adjustStock(bookId, adjustment);

      expect(mockInventoryRepo.recordAdjustment).toHaveBeenCalledWith(
        expect.objectContaining({ reference })
      );
    });

    test('should trigger low stock alert if needed', async () => {
      const bookId = 1;
      const adjustment = { quantity: -50, reason: 'Sales', reference: 'ORD-001' };

      mockBookRepo.findById.mockResolvedValue({
        id: bookId,
        currentStock: 60,
        lowStockThreshold: 100,
      });
      mockInventoryRepo.update.mockResolvedValue({
        id: 1,
        quantity: 10,
      });

      await inventoryService.adjustStock(bookId, adjustment);

      expect(mockAuditService.log).toHaveBeenCalledWith(
        expect.objectContaining({
          action: 'LOW_STOCK_ALERT',
        })
      );
    });
  });
});
```

Expected Results:
✓ Stock increments and decrements working
✓ Negative stock prevention
✓ Audit trails recorded
✓ Low stock alerts triggered
✓ Reference tracking maintained

TEST CASE 8.3: CART SERVICE - TOTAL CALCULATION UNIT TEST
Objective: Test cart total calculation including tax and discounts
Test Type: Unit Test (Isolated)
Scope: CartService.calculateTotal() method
Test Code:
```typescript
describe('CartService', () => {
  let cartService: CartService;
  let mockCartRepo: jest.Mocked<CartRepository>;
  let mockCouponService: jest.Mocked<CouponService>;

  beforeEach(() => {
    mockCartRepo = {
      getItems: jest.fn(),
      find: jest.fn(),
    };
    mockCouponService = {
      validateCoupon: jest.fn(),
      calculateDiscount: jest.fn(),
    };
    cartService = new CartService(mockCartRepo, mockCouponService);
  });

  describe('calculateTotal', () => {
    test('should calculate subtotal from cart items', async () => {
      const cartId = 1;
      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 2, price: 100 },
        { bookId: 2, quantity: 1, price: 50 },
      ]);

      const result = await cartService.calculateTotal(cartId);

      expect(result.subtotal).toBe(250);
    });

    test('should apply 10% tax to subtotal', async () => {
      const cartId = 1;
      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 1, price: 100 },
      ]);

      const result = await cartService.calculateTotal(cartId);

      expect(result.tax).toBe(10);
    });

    test('should apply valid coupon discount', async () => {
      const cartId = 1;
      const couponCode = 'SAVE10';

      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 1, price: 100 },
      ]);
      mockCouponService.validateCoupon.mockResolvedValue(true);
      mockCouponService.calculateDiscount.mockResolvedValue(10);

      const result = await cartService.calculateTotal(cartId, couponCode);

      expect(result.discount).toBe(10);
      expect(result.total).toBe(100);
    });

    test('should reject invalid coupon', async () => {
      const cartId = 1;
      const couponCode = 'INVALID';

      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 1, price: 100 },
      ]);
      mockCouponService.validateCoupon.mockResolvedValue(false);

      await expect(
        cartService.calculateTotal(cartId, couponCode)
      ).rejects.toThrow('Invalid coupon code');
    });

    test('should calculate final total correctly', async () => {
      const cartId = 1;
      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 1, price: 100 },
      ]);

      const result = await cartService.calculateTotal(cartId);

      expect(result.total).toBe(110); // 100 + 10 tax
    });

    test('should handle empty cart', async () => {
      const cartId = 1;
      mockCartRepo.getItems.mockResolvedValue([]);

      const result = await cartService.calculateTotal(cartId);

      expect(result.subtotal).toBe(0);
      expect(result.tax).toBe(0);
      expect(result.total).toBe(0);
    });

    test('should not allow negative discount', async () => {
      const cartId = 1;
      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 1, price: 100 },
      ]);
      mockCouponService.calculateDiscount.mockResolvedValue(-50);

      await expect(
        cartService.calculateTotal(cartId, 'COUPON')
      ).rejects.toThrow('Invalid discount value');
    });

    test('should not allow discount exceeding subtotal', async () => {
      const cartId = 1;
      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 1, price: 100 },
      ]);
      mockCouponService.calculateDiscount.mockResolvedValue(150);

      await expect(
        cartService.calculateTotal(cartId, 'COUPON')
      ).rejects.toThrow('Discount exceeds subtotal');
    });

    test('should return all calculation components', async () => {
      const cartId = 1;
      mockCartRepo.getItems.mockResolvedValue([
        { bookId: 1, quantity: 1, price: 100 },
      ]);

      const result = await cartService.calculateTotal(cartId);

      expect(result).toHaveProperty('subtotal');
      expect(result).toHaveProperty('tax');
      expect(result).toHaveProperty('discount');
      expect(result).toHaveProperty('total');
      expect(result).toHaveProperty('itemCount');
    });
  });
});
```

Expected Results:
✓ Subtotal calculated correctly
✓ Tax applied properly
✓ Discounts validated
✓ Final total accurate
✓ Edge cases handled

UNIT TEST BEST PRACTICES IMPLEMENTED
================================================================================

1. Test Isolation
   ☑ Each test independent
   ☑ No shared state between tests
   ☑ Fresh mocks in beforeEach()
   ☑ Clean test data

2. Clear Test Names
   ☑ Describes what is being tested
   ☑ Expected outcome clear
   ☑ Business logic readable

3. AAA Pattern
   ☑ Arrange: Set up test conditions
   ☑ Act: Execute the test
   ☑ Assert: Verify results

4. Mocking Strategy
   ☑ Mock external dependencies
   ☑ Jest.mock() for modules
   ☑ jest.spyOn() for tracking calls
   ☑ Mock data reflects reality

5. Edge Case Coverage
   ☑ Null/undefined inputs
   ☑ Boundary values
   ☑ Empty collections
   ☑ Large datasets
   ☑ Concurrent operations

6. Error Handling
   ☑ Expected exceptions tested

   ☑ Recovery paths tested
